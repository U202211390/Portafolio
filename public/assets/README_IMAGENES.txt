IMÁGENES NECESARIAS PARA EL PORTFOLIO
======================================

Por favor, agrega las siguientes imágenes en esta carpeta (public/assets/):

1. profile.jpg
   - Tu foto de perfil profesional
   - Tamaño recomendado: 500x500px (cuadrada)
   - Formato: JPG o PNG

2. Imágenes de proyectos:
   - urbanpass.jpg - Captura o logo del proyecto UrbanPass
   - bonofacil.jpg - Captura o logo del proyecto BonoFacil
   - aventurepe.jpg - Captura o logo del proyecto AventurePe
   - hanoi.jpg - Captura o visualización del proyecto Torres de Hanoi
   - Tamaño recomendado: 800x600px
   - Formato: JPG o PNG

3. Imágenes para la tienda:
   - store-api.jpg - Imagen representativa de desarrollo de APIs
   - store-webapp.jpg - Imagen representativa de aplicaciones web
   - store-server.jpg - Imagen de servidor o hardware
   - store-workstation.jpg - Imagen de workstation o PC
   - Tamaño recomendado: 600x400px
   - Formato: JPG o PNG

NOTA: Si no tienes estas imágenes, puedes usar imágenes de stock temporales
de sitios como Unsplash, Pexels o usar placeholders mientras consigues las
imágenes finales.

Las imágenes deben colocarse directamente en la carpeta public/assets/
